Offline Radar Setup

How AI Was Used

AI is integrated for future flight pattern analysis, anomaly detection, and predictive modeling to classify signals and anticipate airspace behavior.

Purpose of the Project

To build a decentralized, offline-capable system that tracks nearby aircraft using ADS-B signals, independent of internet infrastructure.

Problem It Solves

– Increases local airspace transparency
– Bypasses reliance on web-based radar platforms
– Opens up learning opportunities in signal intelligence and radio technology

Applications

– Aviation education and STEM outreach
– Localized airspace monitoring
– Experimental systems for drone air traffic coordination

How I Built It

Hardware:
– RTL-SDR v3 USB dongle
– Standard SMA antenna
– ThinkPad T495 (Zorin OS)
– USB extension cable

Software:
– Dump1090, GQRX, Virtual Radar Server (optional)
– Python scripts (planned)

Steps:
1. Installed Zorin OS
2. Connected RTL-SDR & antenna
3. Installed Dump1090 & GQRX
4. Validated real-time signal capture
5. (Planned) AI analysis layer

Automated Reel Maker

How AI Was Used

AI powers every step — from scraping Reddit to generating scripts, voiceovers, and matching visuals using APIs. Captions are stylized for vertical mobile-first formats.

Purpose of the Project

To fully automate short-form content creation (TikToks, Shorts) with a professional style — no editing required.

Problem It Solves

– Eliminates editing time
– Enables non-creators to publish videos
– Powers consistent content pipelines

Applications

– Niche education/inspiration content
– Marketing automation
– Passive income content loops

How I Built It

Tech Stack:
Python (MoviePy, ElevenLabs, Pexels, OpenAI)

Process:
1. Scrape and summarize content
2. Generate script
3. Voiceover (AI)
4. Pull Pexels video
5. Add captions and compile into vertical format
6. Export MP4

Demo Video

Built in under 3 hours as a challenge to automate full video creation.
